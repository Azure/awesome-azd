## 1. Introduction

The Azure Developer CLI (`azd`) is an open-source tool developer-friendly commands that map to key stages in your workflow (code, build, deploy, monitor) to deploy to Azure. 

Start from an azd application templates that include everything you need to get up and running on Azure, or start from your code base, and make changes from there. Templates include application code and infrastructure as code assets written in Bicep or Terraform and are infused with recommended practices for the cloud. These templates cover end-to-end scenarios that go far beyond "Hello World!".
